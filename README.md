## 🚀 Getting Started

### Prerequisites

- Git
- Your favorite code editor
- Internet connection

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ghostli123/welcome-youtube.git
```

2. Navigate to the project directory:
```bash
cd welcome-youtube
```
